Типы селекторов

Универсальный *

Звёздочка (*) - универсальный селектор для CSS. Соответствует любому тегу. 

            * {
                margin: 0;
                padding: 0;
               }

Селекторы по типу элемента

Этот базовый селектор выбирает тип элементов, к которым будет применяться правило. Пример: селектор h2 выберет все элементы <h2>. 

            h2 {
                color: aqua;
            }

Селекторы по классу

Этот базовый селектор выбирает элементы, основываясь на значении их атрибута class. Пример: селектор .class выберет все элементы с соответствующим c этим классом. В CSS файле нужно написать следующее:

            .class {
                color: red;
            }

Селекторы по идентификатору

Этот базовый селектор выбирает элементы, основываясь на значении их id атрибута. Идентификатор должен быть уникальным. Пример: селектор #main выберет элемент с идентификатором main.

            #main {
                color: green;
            }

Селекторы по атрибуту

Этот селектор выбирает все элементы, имеющие данный атрибут или атрибут с определённым значением.Например: селектор a[href^="#"] выбирает внутрение ссылки, которые не имеют URL-адреса.

Контекстные селекторы

Используется для выбора элементов, удовлетворяющих определённому контексту. Например, чтобы задать стиль ссылкам, которые расположены внутри списка, нужно указать следующее:

            li a {
                color: black;
                }

Использование комбинаторов

Запятая

Комбинатор , это способ группировки нескольких селекторов. Например: .class, #id{color:red; }

Пробел

выбирает элементы, которые находятся внутри указанного элемента вне зависимости от уровня вложенности. Например: div p{background: blue} покрасит фон всех параграфов в тэге div.

Дочерние селекторы

Комбинатор '>' в отличие от пробела выбирает только те элементы, которые являются дочерними непосредственно по отношению к указанному элементу. Пример: селектор ul > li выберет только дочерние элементы <li> , которые находятся внутри, на первом уровне вложенности по отношению к элементу <ul>.

Комбинатор всех соседних элементов

Комбинатор '~' выбирает элементы, которые находятся на этом же уровне вложенности, после указанного элемента, с тем же родителем. Пример: p ~ span выберет все элементы <span>, которые находятся после элемента <p> внутри одного родителя.

Комбинатор следующего соседнего элемента

Комбинатор '+' выбирает элемент, который находится непосредственно после указанного элемента, если у них общий родитель.

Псевдоклассы

Знак : позволяет выбрать элементы, основываясь на информации, которой нет в дереве элементов. Например: a:hover соответствует ссылке, над которой проходит указатель мыши.

Псевдоэлементы

Например: span::first-letter стилизирует только первую букву в элементе <span>.
